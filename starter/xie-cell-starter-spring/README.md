# Xie-Cells Spring Starter 模块

Spring Boot核心功能的Starter，提供Spring Boot应用的基础配置和扩展功能。

## 功能列表

- [自定义日志处理器](./docs/logging-guide.md) - 提供自定义日志处理和文件标记功能

## 使用方式

在您的项目中添加以下依赖：

```xml
<dependency>
    <groupId>io.github.xiechanglei</groupId>
    <artifactId>xie-cell-starter-spring</artifactId>
    <version>3.5.10.1</version>
</dependency>
```
