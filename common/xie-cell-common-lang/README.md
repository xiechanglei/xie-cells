# Xie-Cells Common Lang 模块

这是一个通用语言级别工具模块，提供各种常用的Java语言级别工具类。

## 功能列表

- 字符串工具类 (StringUtils) - 提供字符串判空、截取、格式化等功能

## 使用方式

在您的项目中添加以下依赖：

```xml
<dependency>
    <groupId>io.github.xiechanglei</groupId>
    <artifactId>xie-cell-common-lang</artifactId>
    <version>3.5.9.1</version>
</dependency>
```
